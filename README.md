## How to run

1. Run `npm install`;
2. Put REACT_APP_API_KEY, REACT_APP_APP_NAME, REACT_APP_MESSAGING_SENDER_ID, REACT_APP_APP_ID and REACT_APP_MEASUREMENT_ID in .env file;
3. Run `npm start`.
